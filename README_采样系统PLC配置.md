# 采样系统PLC配置说明

## 系统概述

本系统基于AIR8000芯片实现采样系统的PLC控制功能，支持K1/K5/K9按键输入控制，S0-S3继电器输出，以及通过U1路485与采样探头的Modbus RTU通信。

## 硬件配置

### 输入端口 (K1-K11通过ADC监控)
- **K1**: 启动采样系统
- **K5**: 执行反吹操作  
- **K9**: 执行标定操作

### 输出端口 (S0-S3通过GPIO控制)
- **S0**: 采样泵控制
- **S1**: 标定切换阀
- **S2**: 原位标定切换阀
- **S3**: 超标报警输出

### 通信配置
- **U1路485**: 采样探头通信
- **协议**: Modbus RTU
- **参数**: 9600, 8, N, 1
- **超时**: 2秒
- **设备地址**: 01

## 控制序列

### 1. K1启动序列
```
K1按下 → 启动采样泵 → 发送启动采样指令 → 等待系统稳定(2秒)
```

### 2. K5反吹序列
```
K5按下 → 打开标定切换阀 → 等待阀门动作(1秒) → 发送反吹指令 → 反吹持续10秒 → 停止反吹 → 关闭标定切换阀
```

### 3. K9标定序列
```
K9按下 → 停止采样泵 → 等待采样泵停止(1秒) → 打开原位标定切换阀 → 打开标定切换阀 → 等待阀门动作(2秒) → 发送标定指令 → 标定持续30秒 → 停止标定 → 关闭标定切换阀 → 关闭原位标定切换阀 → 重新启动采样泵
```

## Modbus命令库

| 命令 | 十六进制 | 描述 |
|------|----------|------|
| start_sampling | 01 06 00 0C 00 01 78 08 | 截止阀开启 - 启动采样 |
| stop_sampling | 01 06 00 0C 00 00 B9 C8 | 截止阀关闭 - 停止采样 |
| start_backflush | 01 06 00 0D 00 01 69 C8 | 反吹阀开启 - 开始反吹 |
| stop_backflush | 01 06 00 0D 00 00 A8 08 | 反吹阀关闭 - 停止反吹 |
| start_calibration | 01 06 00 0E 00 01 29 C8 | 标定阀开启 - 开始标定 |
| stop_calibration | 01 06 00 0E 00 00 68 08 | 标定阀关闭 - 停止标定 |
| emergency_stop | 01 06 00 0F 00 00 F9 C8 | 紧急停止 - 关闭所有阀门 |

## 系统状态

- **IDLE**: 空闲状态
- **RUNNING**: 运行状态（采样泵工作）
- **CALIBRATING**: 标定状态
- **BACKFLUSHING**: 反吹状态
- **EMERGENCY**: 紧急状态

## 配置文件

### 主配置文件
- 路径: `/plc_sequences.json`
- 备份文件: `/plc_sequences_backup.json`

### 采样系统配置
- 路径: `/pc_config/sampling_system_config.json`
- 包含预定义的K1/K5/K9控制序列

## 使用方法

### 1. 系统启动
系统启动后会自动加载配置文件，初始化所有硬件模块。

### 2. 按键操作
- 按下K1启动采样系统
- 按下K5执行反吹操作
- 按下K9执行标定操作

### 3. 状态监控
系统会实时显示：
- PLC当前状态
- 活动序列
- 三路温度值
- 开关状态

### 4. 配置修改
可以通过PC端配置工具修改控制序列：
- 运行 `python pc_config/plc_config_ui.py`
- 编辑序列和步骤
- 保存并上传到设备

## 安全特性

### 互锁保护
- 标定时采样泵必须停止
- 反吹和标定不能同时进行
- 紧急停止时关闭所有阀门

### 通信保护
- Modbus通信超时保护（2秒）
- 通信失败时的错误处理
- 发送指令后验证执行结果

## 故障排除

### 常见问题

1. **温度读取异常**
   - 检查MAX31865连接
   - 验证SPI配置参数

2. **开关无响应**
   - 检查ADC通道配置
   - 验证去抖时间设置

3. **Modbus通信失败**
   - 检查485连接
   - 验证通信参数
   - 检查设备地址

4. **序列执行异常**
   - 检查配置文件格式
   - 查看系统日志
   - 验证GPIO引脚配置

### 日志查看
系统会记录详细的操作日志，包括：
- 系统启动/停止
- 状态转换
- 命令执行结果
- 错误信息

## 技术规格

- **主控芯片**: AIR8000
- **温度精度**: ±0.5°C (PT100)
- **响应时间**: <100ms
- **工作温度**: -40°C ~ +85°C
- **电源电压**: 3.3V
- **通信接口**: UART, SPI, I2C, GPIO
