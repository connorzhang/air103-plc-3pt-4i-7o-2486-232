# 采样系统PLC实现状态总结

## 🎯 当前实现状态

### ✅ 已完成功能

#### 1. K1采样控制功能
- **功能描述**: 启动/停止采样泵（S0继电器）
- **控制逻辑**: 按键按下启动，松开停止
- **安全保护**: 与标定和反吹互锁
- **状态管理**: 完整的运行状态跟踪

#### 2. K5反吹控制功能
- **功能描述**: 执行反吹操作（S1继电器）
- **控制序列**: 
  1. 打开标定切换阀
  2. 等待阀门动作（1秒）
  3. 发送反吹指令
  4. 反吹持续10秒
  5. 停止反吹
  6. 关闭标定切换阀
- **安全保护**: 与标定互锁，自动超时保护

#### 3. K9标定控制功能
- **功能描述**: 执行标定操作（S1+S2继电器）
- **控制序列**:
  1. 停止采样泵
  2. 等待采样泵停止（1秒）
  3. 打开原位标定切换阀（S2）
  4. 打开标定切换阀（S1）
  5. 等待阀门动作（2秒）
  6. 发送标定指令
  7. 标定持续30秒
  8. 停止标定
  9. 关闭标定切换阀
  10. 关闭原位标定切换阀
  11. 重新启动采样泵
- **安全保护**: 与采样和反吹互锁，自动超时保护

#### 4. 基础架构
- **开关监控**: 完整的K1-K11开关状态检测（ADC监控）
- **GPIO控制**: 10路继电器输出控制
- **状态管理**: 系统状态机（IDLE/RUNNING/CALIBRATING/BACKFLUSHING）
- **任务调度**: 基于sys.taskInit的并发任务管理
- **日志系统**: 详细的操作日志记录

### 🔧 技术特性

#### 安全互锁系统
- **采样泵互锁**: 标定时自动停止，标定完成后自动重启
- **功能互斥**: 标定、反吹、采样不能同时进行
- **超时保护**: 反吹15秒超时，标定60秒超时
- **紧急停止**: 一键停止所有操作，关闭所有阀门

#### Modbus RTU通信
- **命令库**: 完整的采样探头控制指令
- **通信参数**: 9600, 8, N, 1, 2秒超时
- **设备地址**: 01
- **模拟实现**: 当前为模拟通信，可替换为实际UART1 485

#### 状态监控
- **实时状态**: 2秒间隔的状态更新
- **安全状态**: 安全互锁状态查询
- **操作历史**: 完整的操作日志记录

## 🚀 使用方法

### 1. 系统启动
```lua
-- 主程序自动启动所有功能
require("plc_controller").run()
```

### 2. 按键操作
- **K1**: 启动/停止采样泵
- **K5**: 启动/停止反吹操作
- **K9**: 启动/停止标定操作

### 3. 手动控制（测试用）
```lua
local plc = require("plc_controller")

-- 手动控制采样泵
plc.manual_start_sampling()
plc.manual_stop_sampling()

-- 手动控制反吹
plc.manual_start_backflush()
plc.manual_stop_backflush()

-- 手动控制标定
plc.manual_start_calibration()
plc.manual_stop_calibration()

-- 紧急停止
plc.emergency_stop()
```

### 4. 状态查询
```lua
-- 获取系统状态
local status = plc.get_system_status()
print("当前状态:", status.current_state)

-- 获取安全状态
local safety = plc.get_safety_status()
print("安全锁定:", safety.safety_locked)
```

## 📊 性能指标

- **响应时间**: <100ms（按键响应）
- **状态更新**: 2秒间隔
- **安全检查**: 100ms间隔
- **开关防抖**: 10ms

## 🛡️ 安全特性

- **硬件互锁**: 继电器状态互斥
- **软件互锁**: 功能状态互斥
- **超时保护**: 自动停止超时操作
- **紧急停止**: 一键停止所有功能
- **状态验证**: 操作前后状态检查

4. **日志记录**: 所有操作都有详细日志
